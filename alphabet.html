The Amharic Alphabet

The Ge’ez Script

Amharic is written using the Ge’ez script, known as the fidel. Each character represents a consonant paired with a vowel sound, so instead of memorizing an alphabet like A-B-C, learners memorize a grid of consonant-vowel combinations. Below is a reference chart of the core fidel characters.
